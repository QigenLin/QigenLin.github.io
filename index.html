我的博客

发表于 2017-12-25 | 分类于 个人博客

欢迎来到我的个人博客。我是一名在读自然灾害学博士生，研究兴趣是利用 R 和 Python 进行自然灾害时空建模和风险评估。

在我的同学李亚飞和我加入的知识星球版主的影响下，我最近想开始写博客，按照目前的构想我博客的内容主要包括以下几个方面，可能会有越来越多想分享的内容。

  • 读博士过程中一些学习和思考的个人感想；
  • 研究方向上的专业知识介绍，包括最新英文文章阅读以及国外博客；
  • 是利用 R 和 Python 进行自然灾害研究的技术分享；

希望我能坚持下来，通过博客记录自己学习和成长过程，在此基础上做我力所能及的一些分享，不断进步和提高。
