2017年个人总结与回顾

发表于 2017-12-30 | 分类于 个人博客 ， 总结

不知不觉间，2017 年也即将过完了。这是在北京读研的第 4 年，想着要多记录下来学习和生活过程中的事情，就从这篇年终回顾总结开始吧。

学习和科研

从作为研究生的主业开始总结，对自己今年在学习和科研成果上总体比较满意。发表了自己第一篇 SCI 论文，这篇论文研一的时候做出来的一个成果，虽然一直不太满意，但是发表出来也算是完成了一件事。在地理学报上发表了一篇论文。这篇论文讲起来各种心酸史，从开始做到最终发表刊出前后历时 3 年，中间经历了多次拒稿多次修改重写，真的体会到做研究写论文的不易。将邹师兄遥感滑坡提取和单体划分的成果整理完成。滑坡提取部分发表在中文期刊遥感技术与应用，单体划分部分发表在第四届世界滑坡大会会议论文集上。这也让我有机会独自一人赴斯洛文尼亚参加国际会议。第一次在国外滑坡研究学者同行面前做了 15 分钟的报告，当然很紧张，还有许多不足主要继续学习。在小范围研究同行中，与几位国际学者进行了交流，认识了港大、南大和地质大学的几位同学。滑坡大会上体会比较深的是，在这种小同行的会议上，相比起去年的 AGU，能有更多有效的科研交流。有一些老学者特别乐于分享自己的研究经历和人生经验，关键是需要自己踏出第一步，多找不同的人聊天。学术会议之外，这也是一次个人旅游，卢布尔雅那这个城市很漂亮，虽然很小，但是沿河而居的环境、各式的建筑、充满欢乐与自由人文气息，无不令人感觉非常舒适与享受。非常值得再去的地方。下半年的另一篇中文论文也在这两天刊出，还有做河北气象局项目申请了一项软件著作权，参与编制教材的内容还需要继续完善。

自主学习上，今年春节报了一门数据科学实训营的网络课。坚持下来了每次的课程学习，基本把利用 Python 进行数据获取、分析、可视化和建模的流程实操了一遍，掌握了另一项新技能。想想有时候报这种课程也挺不错的，花点钱节省时间，按着有人规划好的学习方法走下来就行，比自己自学要省心一些。

关于如果提出科学问题，做好一个研究这方面的思考，还有许多要提高的。在今年准备开题报告和研究计划的过程中，慢慢体会学习和锻炼。

人际沟通

在和导师的交流过程中，发现了许多值得学习的地方。一方面做好自己手头该做的、无法推掉的事情，尽量导师满意，另一方面得有自己的计划和方向，自己做好准备。此外，能有换位思考或者说是乙方思维，能够让自己做的事情更让人认可。这体现在非常多的小细节上：具体有林乐同学给导师和黄老师发的邮件上的细节。stormzhang 知识星球中提到的给他的留言手机号输入的顺序。多提别人考虑一步，自己也能得到更多的认可。参与河北省气象局项目的几次汇报中发现，地方上喝酒真的也是一种能力，而且是进入工作以后非常必要的能力。

学术交流上，可以主动一些与国外同行进行交流，有问题就多发邮件咨询。就写一个邮件也费不了多大事情，如果能收到回复那就是非常大的收获了。

IT圈子

最近加入了一个 IT 方向 stormzhang 的知识星球，想了解一些 IT 行业相关的职场问题和其他方面的相关分享，知道多一些工作职业的问题，保持多一点可能性。

临时想开始写，总感觉不知道从哪着手，不知道要写什么内容，写出来也像流水账一般混乱。明年会是挺关键的一年，希望接下来一年自己能保持沉着的心态，能有持续的学习热情，一步步实现目标。

我的博客

发表于 2017-12-25 | 分类于 个人博客

欢迎来到我的个人博客。我是一名在读自然灾害学博士生，研究兴趣是利用 R 和 Python 进行自然灾害时空建模和风险评估。

在我的同学李亚飞和我加入的知识星球版主的影响下，我最近想开始写博客，按照目前的构想我博客的内容主要包括以下几个方面，可能会有越来越多想分享的内容。

  • 读博士过程中一些学习和思考的个人感想；
  • 研究方向上的专业知识介绍，包括最新英文文章阅读以及国外博客；
  • 利用 R 和 Python 进行自然灾害研究的技术分享；

希望我能坚持下来，通过博客记录自己学习和成长过程，在此基础上做我力所能及的一些分享，不断进步和提高。

Qigen Lin
